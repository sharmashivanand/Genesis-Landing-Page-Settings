== Description ==

Landing Page Settings for Genesis plugin allows you to easily show / hide header, footer, breadcrumbs and other elements on a page, post or any custom post type on your Genesis powered website. This plugin exclusively works with Genesis framework. No more tinkering with functions.php; just configure the plugin options to update. You can change the settings globally or individually on each post, page.

You can configure the settings for the desktop and mobile viewport independently.


== Installation ==

Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New. In the search field type “Genesis Landing Page Settings” and click Search Plugins. Once you’ve found the plugin you can install it by simply clicking “Install Now”.

Or you can follow the steps given below:

1. Upload the entire `genesis-landing-page-settings` folder to the `/wp-content/plugins/` directory.
1. DO NOT change the name of the `genesis-landing-page-settings` folder.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Set up the options as required.
1. Save the changes.
